# Procurement Analysis Skill Candidate

## Purpose

Support purchasing analysis, supplier comparison, raw-material market review, quote normalization, and cost-change explanation.

## Expected Inputs

- Product or material scope
- Supplier data
- Price history
- Quote details
- Market assumptions

## Expected Outputs

- Cost analysis
- Supplier comparison
- Risk notes
- Recommendation summary

## Creation Trigger

Create this as a real skill only after repeated procurement-analysis work needs the same workflow, validation rules, and output structure.
