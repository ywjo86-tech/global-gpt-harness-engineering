# AI Agent Development Skill Candidate

## Purpose

Support AI agent workflow design, prompt and tool contracts, evaluation plans, and OpenAI or Codex-based implementation patterns.

## Expected Inputs

- Agent goal
- User workflow
- Tool or API requirements
- Evaluation criteria
- Safety constraints

## Expected Outputs

- Agent architecture
- Tool contract
- Prompt or instruction design
- Evaluation plan
- Deployment notes

## Creation Trigger

Create this as a real skill only after repeated agent-development work needs durable design and QA standards.
