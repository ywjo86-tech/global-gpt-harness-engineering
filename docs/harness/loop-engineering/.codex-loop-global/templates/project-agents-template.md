# 프로젝트 AGENTS 템플릿

## 상속

이 프로젝트는 `AGENTS.md`와 `LOOP_ENGINEERING_POLICY.md`의 루트 Loop Engineering 정책을 상속한다.

## 책임

- 모든 코드 작업에 루프 절차를 적용한다.
- 수정 전에 평가 기준을 고정한다.
- 루프당 하나만 변경한다.
- 성공과 실패를 모두 기록한다.
- 실패한 접근 방식을 재사용하지 않는다.

## 서브에이전트 경계

- Frontend 서브에이전트: UI, 레이아웃, 스타일, 인터랙션.
- Backend 서브에이전트: API, 서버 로직, 데이터 처리, 인증, 백엔드 오류.
- QA 서브에이전트: 검증, 회귀 확인, 완료 게이트.
- Logger 서브에이전트: 루프 기록 관리.
- Test 서브에이전트: 테스트 분석과 테스트 중심 수정.
- Refactor 서브에이전트: 구조와 유지보수성 개선.
