# Project Harness Template

Use this template as a starting point for a Codex-managed project repository.

## Replace

- Project name
- Business purpose
- Build and test commands
- Repository-specific paths
- Release and deployment notes

## Include

- A concise `AGENTS.md`
- Requirements document
- Design document
- Test plan
- Release handoff

## Exclude by Default

- `_workspace/`
- Secrets
- Raw customer data
- Raw email content
- Confidential procurement data
